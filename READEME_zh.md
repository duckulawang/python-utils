# utils

实用工具与脚本集合。

---

## 🛠️ 工具列表

### 📐 Function Plot
一个可直接在浏览器中运行的交互式函数图像生成器。  
支持显函数 `y=f(x)`、参数方程和隐函数 `f(x,y)=0`。  
可平滑缩放、拖拽平移，并在曲线上实时追踪坐标。

👉 **[打开 Function Plot](./FunctionPlot/FunctionPlot.html)**

---

### 🖼️ raw_selector
一个用于快速浏览和挑选 RAW 图像文件的 Python 桌面应用。  
（需要在本地安装 Python 运行，非在线工具。）

📁 源码目录：[`raw_selector/`](./raw_selector/)
