# 函数图像生成器 — 支持类型与示例

本工具基于 **math.js** 数学库，支持其内置的全部初等函数、常量及运算符。以下按绘图模式分类说明，并给出可直接输入的表达式的示例。

---

## 一、显函数模式 `y = f(x)`

在该模式下，您只需输入右侧表达式（包含变量 `x`），即可绘制 `y = f(x)` 的图像。

### 1. 基本运算符
| 运算符 | 含义 | 示例表达式 |
|--------|------|------------|
| `+` `-` `*` `/` | 加减乘除 | `2*x + 1` |
| `^` | 乘方 | `x^3 - 2*x^2 + 1` |
| `%` | 取模 | `x % 2` |
| `!` | 阶乘 | `x!` （仅整数） |

### 2. 常用数学函数
| 函数 | 说明 | 示例表达式 |
|------|------|------------|
| `sin(x)` | 正弦 | `sin(x)` |
| `cos(x)` | 余弦 | `cos(2*x)` |
| `tan(x)` | 正切 | `tan(x)` |
| `asin(x)` | 反正弦 | `asin(x)` |
| `acos(x)` | 反余弦 | `acos(x)` |
| `atan(x)` | 反正切 | `atan(x)` |
| `sinh(x)` | 双曲正弦 | `sinh(x)` |
| `cosh(x)` | 双曲余弦 | `cosh(x)` |
| `exp(x)` | e 的 x 次方 | `exp(x)` |
| `log(x)` | 自然对数 | `log(x)` |
| `log10(x)` | 常用对数 | `log10(x)` |
| `sqrt(x)` | 平方根 | `sqrt(x)` |
| `abs(x)` | 绝对值 | `abs(x)` |
| `sign(x)` | 符号函数 | `sign(x)` |
| `ceil(x)` | 向上取整 | `ceil(x)` |
| `floor(x)` | 向下取整 | `floor(x)` |
| `round(x)` | 四舍五入 | `round(x)` |

### 3. 常数
- `pi` — 圆周率 π（3.14159...）
- `e` — 自然常数（2.71828...）
- `i` — 虚数单位（仅支持复数表达式，但绘图仅显示实数部分）

示例：
```
sin(pi * x)
exp(-x^2)
log(abs(x + 1))
```

### 4. 分段函数（使用 `piecewise`）
格式：`piecewise([条件1, 表达式1], [条件2, 表达式2], ...)`

条件中可用比较运算符：`<`, `<=`, `>`, `>=`, `==`, `!=`。

| 示例 | 表达式 |
|------|--------|
| 绝对值函数 | `piecewise([x < 0, -x], [x >= 0, x])` |
| 阶梯函数 | `piecewise([x < -1, -1], [x < 1, 0], [x >= 1, 1])` |
| 复合分段 | `piecewise([x < 0, -x^2], [x < 2, x], [x >= 2, sqrt(x)])` |

**注意**：条件必须覆盖所有情况，否则未覆盖区域可能报错。您可用最后一个条件作为“否则”分支，例如 `[x >= 2, ...]`。

## 二、参数方程模式

该模式下，您需要分别输入 `x(t)` 和 `y(t)` 的表达式，并指定参数 `t` 的范围。

**变量名**：`t`

| 图形 | x(t) 表达式 | y(t) 表达式 | t 范围 |
|------|-------------|-------------|--------|
| 圆 | `cos(t)` | `sin(t)` | `0` 到 `2*pi` |
| 椭圆 | `2*cos(t)` | `sin(t)` | `0` 到 `2*pi` |
| 抛物线 | `t` | `t^2` | `-5` 到 `5` |
| 摆线 | `t - sin(t)` | `1 - cos(t)` | `0` 到 `4*pi` |
| 螺旋线 | `t*cos(t)` | `t*sin(t)` | `0` 到 `10*pi` |
| 李萨如图形 | `sin(3*t)` | `cos(5*t)` | `0` 到 `2*pi` |

所有支持 `x` 的数学函数同样可用于 `t`，例如 `sin(t)`, `exp(t)`, `log(t)` 等。

## 三、隐函数模式（方程 `f(x, y) = 0`）

该模式下，您输入一个包含 `x` 和 `y` 的方程，工具将绘制所有满足 `f(x, y) = 0` 的点。

### 示例
| 图形 | 表达式 |
|------|--------|
| 圆 | `x^2 + y^2 - 1` |
| 椭圆 | `x^2/4 + y^2 - 1` |
| 双曲线 | `x^2 - y^2 - 1` |
| 抛物线 | `y - x^2` |
| 直线 | `y - 2*x + 1` |
| 三次曲线 | `y^2 - x^3 + x` |
| 三角函数曲线 | `sin(x) - y` |
| 复杂隐式曲线 | `y^2 - x^2 + x^4` |
| 纽结曲线 | `y^2 - x^2 * (1 - x^2)` |

**注意**：
- 隐函数使用 **Marching Squares** 算法，在可视区域内逐像素采样，因此精细的细节需要足够的分辨率（自动调整）。
- 表达式必须能对任意 `(x, y)` 求值，请避免分母为零等无定义情况（可用 `piecewise` 构造条件，但隐函数模式下使用复杂分段可能较慢）。

## 四、常见错误与提示

1. **变量名错误**  
   显函数模式只能用 `x`；参数方程用 `t`；隐函数用 `x` 和 `y`。其他变量名会导致 `undefined` 错误。

2. **括号不匹配**  
   确保 `sin(x` 补全为 `sin(x)`。

3. **函数名大小写敏感**  
   `Sin(x)` 无效，必须使用 `sin(x)`。

4. **分段函数遗漏条件**  
   若 `piecewise` 的条件未覆盖某段 `x` 值，该点计算会返回 `undefined`，图像断开。

5. **隐函数性能**  
   若表达式过于复杂，重绘可能稍慢。可适当缩小视图范围以减少计算量。
