# utils

A collection of practical tools and scripts.

---

## 🛠️ Tools

### 📐 Function Plot
An interactive function graph generator that runs directly in your browser.  
Supports explicit functions `y=f(x)`, parametric equations, and implicit equations `f(x,y)=0`.  
Features smooth zooming, panning, and live coordinate tracking on the curve.

👉 **[Open Function Plot](https://duckulawang.github.io/utils/FunctionPlot/FunctionPlot.html)**

---

### 🖼️ raw_selector
A Python desktop application for quickly browsing and selecting RAW image files.  
(Requires Python to run locally – not a web tool.)

📁 Source: [`raw_selector/`](./raw_selector/)
