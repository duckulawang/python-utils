# Raw Selector

> 这个名字源于最初的使用场景：给小姐姐拍了一堆照片并返图后，小姐姐挑了一些喜欢的图想要精修（告诉我文件名或者发那些jpg原图），然后我需要从一堆图中找出那些RAW原图。一张张挑很麻烦，特别是要挑几百张的时候，因此我做了这个小脚本。虽然本工具意图是挑图片，但本工具只是搜索文件名，并没有限制后缀名，因此可以处理**任意类型**的文件——文本、视频、压缩包、PDF 等。

一个简单的桌面小工具，用于从源文件夹中**批量移动文件**到目标文件夹。

支持两种筛选方式：

1. **文本筛选**：在文本框中输入文件名列表（每行一个），程序自动在源文件夹的文件列表中选中这些文件。
2. **文件夹筛选**：选择一个参考文件夹，程序读取该文件夹中所有文件的**主文件名（不含扩展名）**，然后在源文件夹的文件列表中自动选中同名的文件（不限制扩展名）。

## 适用场景

- 从大量文件中，根据一个小参考集合（如已整理出的部分文件），快速选中并移动其他格式的同名文件。
- 按照手动整理的文件名清单，批量移动文件。
- 典型原始场景：根据之前挑选出的 JPG 小图，从几千张照片中批量移动对应的 RAW 原图。

## 功能特性

- **移动模式**（源文件会被移动到目标文件夹，不在原位置保留）。
- 图形化界面，所有路径均可通过按钮选择。
- 筛选方式：
  - **文本模式**：在文本框中输入文件名（每行一个，可带扩展名或不带），点击“通过文本筛选”按钮，程序自动在下方文件列表中选中匹配的文件。
  - **文件夹模式**：点击“通过文件夹筛选”按钮，选择一个参考文件夹，程序读取其中所有文件的**主文件名（不含扩展名）**，然后在源文件夹的文件列表中选中所有同名的文件（不限扩展名）。
- 文件列表显示源文件夹中所有文件的主文件名（不含扩展名），支持多选（可额外手动调整选中）。
- 实时日志显示（最多保留 10 条记录）。
- 移动完成后，文件列表自动刷新。

## 安装与运行

### 环境要求
- Python 3.6+
- 依赖库：仅使用 Python 标准库（`tkinter`、`os`、`time`），无需额外安装。

### 步骤
1. 克隆或下载本仓库。
2. 在终端中运行：
   ```bash
   python raw_selector.py
   ```

## 使用方法

1. 点击 **“选择源文件夹”**，选择包含原始文件的文件夹。
2. 点击 **“选择目标文件夹”**，选择文件要移动到的目标位置。
3. **筛选文件**（二选一）：
   - **通过文件夹筛选**：点击按钮，选择一个参考文件夹，程序自动选中源文件夹中与参考文件同名的所有文件（忽略扩展名）。
   - **通过文本筛选**：在下方的文本框中输入文件名（每行一个），然后点击按钮，程序自动选中匹配的文件。
4. 如果需要，可以手动在文件列表中增减选中项（支持多选）。
5. 点击 **“移动文件”**，程序将选中的文件从源文件夹移动到目标文件夹。
6. 操作完成后，文件列表会刷新，日志区会显示移动记录。

## 常见问题

**Q: 为什么叫 Raw Selector 却又说支持任意文件类型？**  
A: 这个工具最初是为摄影师设计的（从 JPG 小图反找 RAW 原图），因此取了这个名字。但它的核心逻辑是基于“主文件名匹配”，不依赖扩展名，因此完全可以用于任何类型的文件。

**Q: 移动后源文件还在吗？**  
A: 不在。程序使用 `os.rename` 执行移动，相当于“剪切”。请确保目标文件夹正确，避免数据丢失。

**Q: 文件夹模式下，程序会读取参考文件夹里所有文件吗？**  
A: 是的，读取所有文件的主文件名（不含扩展名）。如果你只想用特定类型的文件做参考，可以手动过滤参考文件夹的内容。

**Q: 文件名匹配时是否包含扩展名？**  
A: 不包含。程序只比较主文件名。例如参考文件夹中的 `photo.jpg` 会匹配源文件夹中的 `photo.raw`、`photo.NEF`、`photo.mp4`、`photo.txt` 等所有同名文件。

**Q: 如果目标文件夹已存在同名文件会怎样？**  
A: `os.rename` 在 Windows 下如果目标文件已存在会报错。程序没有特殊处理，建议确保目标文件夹中没有同名文件，或先清空。
