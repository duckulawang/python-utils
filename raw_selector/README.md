# Raw Selector

> 这个名字源于最初的使用场景（从照片中批量找出 RAW 原图），但本工具没有区分文件后缀名，因此可以处理**任意类型**的文件——文本、视频、压缩包、PDF 等，核心是根据文件名列表进行筛选。

一个简单的桌面小工具，用于从源文件夹中**批量筛选文件**（复制或移动）到目标文件夹。

支持两种筛选方式：

1. **文本筛选**：提供一个回车分隔的文件名列表，程序从源文件夹中匹配这些文件（支持带扩展名或不带扩展名）并转移。
2. **文件夹筛选**：提供一个“参考文件夹”，程序自动读取该文件夹中**所有文件的主文件名**（忽略扩展名），然后在源文件夹中找出**同名且任意扩展名**的文件并转移。

> 例如：参考文件夹里有一张 `example.jpg`，程序会从源文件夹中找出 `example.raw`、`example.mp4`、`example.pdf` 等所有同名的文件（不限类型）。

## 适用场景

- 从大量文件中，根据一个小参考集合（如已整理出的部分文件），批量提取出其他格式的同名文件。
- 按照一个文件名清单（手动整理的列表）批量收集文件。
- 典型原始场景：根据之前挑选出的 JPG 小图，从几千张照片中批量复制对应的 RAW 原图。

## 功能特性

- 支持 **复制** 或 **移动** 两种操作模式。
- 源文件夹、目标文件夹、筛选方式均可通过图形界面选择。
- 筛选方式：
  - **文本模式**：粘贴或导入文件名列表（每行一个文件名，可带扩展名也可不带）。
  - **文件夹模式**：选择任意一个包含参考文件的文件夹，程序读取其中所有文件的**主文件名（不含扩展名）**，然后在源文件夹中匹配同名的**所有文件（不限扩展名）**，并进行转移。
- 实时显示处理进度和结果统计（成功/失败/跳过）。
- 支持处理子文件夹（可选是否递归搜索源文件夹）。

- 依赖库：`tkinter`（Python 自带）、`os`, `shutil` 等标准库（无需额外安装）。

1. 克隆或下载本仓库。
2. 运行程序：
